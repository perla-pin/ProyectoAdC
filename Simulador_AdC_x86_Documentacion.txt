El programa realiza tres operaciones fundamentales:
    1.- Captura n cantidad de elementos que son enteros proporcionados por el usuario.
    2.- Almacena los valores en un arreglo global de tamaño fijo (256 enteros)
    3.- Consulta un elemento específico del arreglo usando su índice

    Codigo de alto nivel:
        #include <iostream>
        using namespace std;

        int lista[256]; //

        int interrupcion_teclado() {
            int dato;
            cin >> dato;
            return dato;
        }

        int main() {
            int n;
            cout << "Cuantos elementos? ";
            n = interrupcion_teclado();

            for (int i = 0; i < n; i++) {
                cout << "Dato [" << i + 1 << "]: ";
                lista[i] = interrupcion_teclado();
            }

            cout << "\nLista completa:\n";
            for (int i = 0; i < n; i++) {
                cout << "[" << i << "] = " << lista[i] << "\n";
            }

            int idx;
            cout << "\nQue indice consultar? ";
            idx = interrupcion_teclado();
            cout << "Valor en [" << idx << "] = " << lista[idx] << "\n";

            system("pause");

            return 0;
        }

ARQUITECTURA SIMULADA
===============================================================================

El sistema simulado contiene los componentes principales de una
arquitectura Von Neumann.

Componentes principales:

• CPU (Unidad Central de Procesamiento)
• Memoria RAM
• Registros del procesador
• Stack (pila)
• Unidad de Control
• ALU (Unidad Aritmético Lógica)

En esta arquitectura, tanto los datos como las instrucciones se
almacenan en la misma memoria RAM. La CPU obtiene las instrucciones
desde la memoria, las decodifica y posteriormente las ejecuta.

Este proceso se conoce como el ciclo de instrucción.
